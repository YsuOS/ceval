# ceval: the eval cmd from Crash-utility on bash
## Example
```
$ ceval 255
hexdecimal: ff
   decimal: 255
     octal: 377
    binary: 0000000000000000000000000000000000000000000000000000000011111111
  bits set: 7 6 5 4 3 2 1 0
```
```
$ ceval 0xff
hexdecimal: ff
   decimal: 255
     octal: 377
    binary: 0000000000000000000000000000000000000000000000000000000011111111
  bits set: 7 6 5 4 3 2 1 0
```
